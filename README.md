# Fresh Canteen

This project is a standalone frontend application for Fresh Canteen.

## Getting Started

### Prerequisites
- Node.js (v18 or later)
- npm or bun

### Installation
```bash
npm install
# or
bun install
```

### Development
```bash
npm run dev
# or
bun dev
```

### Build
```bash
npm run build
# or
bun run build
```

## Tech Stack
- React
- Vite
- Tailwind CSS
- Shadcn/UI
- Lucide React
- TanStack Query
